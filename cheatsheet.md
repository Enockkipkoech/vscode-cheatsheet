### VSCODE SHORTCUTS CHEATSHEET

#### General

1. Ctrl+Shift+P, F1 - Show command palette.
2. Ctrl+P - Quick Open, Go to File... ( type name of file )
3. Ctrl+Shift+N - New window/ New instance
4. Ctrl+Shift+W - Close window / instance
5. Ctrl+, - User settings
6. Ctrl+K Ctrl+S - Keyboard shortcuts

#### Basic Editing

- Ctrl+X - cut line / empty selecton
- Ctrl+C - Copy line (empty selection)
- Alt+ ‚Üë / ‚Üì - Move line up/down
- Shift+Alt + ‚Üì / ‚Üë - Copy line up/down
- Ctrl+‚Üë / ‚Üì - Scroll line up/down

- Ctrl+Shift+K -Delete line
- Ctrl+Shift+Enter - Insert line above
- Ctrl+Shift+\ - Jump to matching bracket
- Ctrl+Enter - Insert line below
- Ctrl+] / [ - Indent/outdent line
- Home / End - Go to beginning/end of line
- Alt+PgUp / PgDn - Scroll page up/down
- Ctrl+Shift+[ - Fold (collapse) region
- Ctrl+Shift+] - Unfold (uncollapse) region

- Ctrl+K Ctrl+[ - Fold (collapse) all subregions <!-- CAVEAT: Press ctrl+k then release both and press ctrl + [ as second command  -->
- Ctrl+K Ctrl+] - Unfold (uncollapse) all subregions <!-- CAVEAT: Press ctrl+k then release both and press ctrl + ] as second command -->
- Ctrl+K Ctrl+0 - Fold (collapse) all regions
- Ctrl+K Ctrl+J - Unfold (uncollapse) all regions
- Ctrl+K Ctrl+C - Add line comment SAME as ctrl + /
- Ctrl+K Ctrl+C - Remove line comment SAME as ctrl + /
- Ctrl+U - Undo last cursor operation
- Ctrl+/ - Toggle line comment
- Alt+Z - Toggle word wrap

#### Navigation

- Ctrl+T - Show all Symbols
- Ctrl+G - Go to Line...
- Ctrl+P - Go to File...
- Ctrl+Shift+O - Go to Symbol...
- Ctrl+Shift+M - Show Problems panel
- F8 - Go to next error or warning
- Shift+F8 - Go to previous error or warning
- Ctrl+Shift+Tab - Navigate editor group history

#### Search and Replace

- Ctrl+F - Find
- Ctrl+H - Replace
- F3 / Shift+F3 - Find next/previous
- Alt+Enter - Select all occurences of Find match ALMOST same as ctrl+D
- Ctrl+D - Add selection to next Find match
- Ctrl+K Ctrl+D - Move last selection to next Find match
- Alt+C / R / W -Toggle case-sensitive / regex / whole word

#### Multi-cursor & selection

- Alt+Click - Insert cursor
- Ctrl+Alt+ ‚Üë / ‚Üì - Insert cursor above / below
- Ctrl+U - Undo last cursor operation
- Shift+Alt+I - Insert cursor at end of each line selected
- Ctrl+L - Select current line
- Ctrl+Shift+L - Select all occurrences of current selection
- Ctrl+F2 - Select all occurrences of current word
- Shift+Alt+‚Üê - Shrink selection
- Shift+Alt+‚Üí - Expand selection
- Shift+Alt + (drag mouse) - Column (box) selection
- Ctrl+Shift+Alt + (arrow key) - Column (box) selection
- Ctrl+Shift+Alt + PgUp/PgDn - Column (box) selection page up/down
